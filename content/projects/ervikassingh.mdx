## Overview
ervikassingh.com is my personal portfolio website, designed to showcase my projects and provide a way to get in touch. The site is built with modern web technologies and emphasizes clean design, accessibility, and performance.

## Features
- Project portfolio with detailed pages
- Contact information for direct communication
- Animated particle background for visual appeal
- Fully responsive and mobile-friendly

## Technologies Used
- **Next.js** for server-side rendering and routing
- **React** for building interactive UI components
- **TypeScript** for type safety
- **Tailwind CSS** for utility-first styling
- **Contentlayer** for content management with MDX support

If you like the design or want to see how it works, check out the repository on GitHub. Contributions and feedback are always welcome!
